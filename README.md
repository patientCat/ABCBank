# ABCBank
学习ABCBank做的一个银行系统，算是目前将已经学到的所有东西做了整合和磨练。

## 介绍
ABCBank是一个银行存储系统，采用CS架构。
Client端使用wxWidget做的，但是没有做到跨平台，网络通信方面依然使用的是linux下的。
Server端使用的是muduo库。算是学习了muduo库后最终的一个交代。

1. client——客户端文件夹
2. public——共用文件夹
3. server——服务器文件夹
4. mysql——mysql表
